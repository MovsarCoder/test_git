Чтобы добавить .git к нашему проекту
>>> git init 
Если все успешно мы видим
>>> Initialized empty Git repository
Второй раз когда мы введем git init
>>> Reinitialized empty Git repository


Команда чтобы добавить файлы в индекс
>>> git add .

Сохранить коммит (сделать снимок проекта)
>>> git commit -m "Название Коммита"

Команда чтобы отправить изменения во второй и третий и так далее раз в гитхаб а в первый раз используем последние 3 комманды с нового репо
>>> git push

Чтобы получить изменения с гитхаба 
>>> git pull

Чтобы увидеть все ветки нашего гита с нашим проектом
>>> git branch

Создание новой ветки
>>> git branch movsar
>>> git branch Iam7mail

Чтобы заполучить изменения с другой ветки 
я слияю эту ветку с моей
я захожу в ветку в которую мне нужно слиять 
git checkout movsar
git merge main


Чтобы отправить ветки на гитхаб
>>> git push -u origin movsar
>>> git push -u origin Iam7mail
